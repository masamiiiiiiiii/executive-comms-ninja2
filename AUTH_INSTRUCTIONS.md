
# 認証手順 (Authentication Steps)

Google Cloudのモデル(Gemini)を使用するために、以下のコマンドをご自身のターミナルで実行してください。
(To use Gemini, please run the following command in your terminal.)

```bash
gcloud auth application-default login
```
(ブラウザが開き、認証画面が表示されます / A browser will open for login)

もしブラウザが開かない場合 (If browser doesn't open):
```bash
gcloud auth application-default login --no-launch-browser
```

認証が完了したら、「完了しました」と教えてください。
(After authentication, please tell me "Done".)
